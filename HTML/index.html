¿Que es HTML?

HTML(HyperText Markup Language) es un lenguaje de etiquetas. Indica al navegador como tiene que mostrar el contenido. El HTML separa el "contenido" (palabras, imágenes, audio, video, etc.) de la "presentación" (la definición del tipo de contenido y las instrucciones de cómo esos contenidos tienen que mostrarse). El HTML emplea un conjunto de elementos predefinidos que permiten identificar los distintos tipos de elementos. Estos elementos contienen una o más etiquetas que contienen o expresan el contenido. Estas etiquetas suelen ir encapsuladas entre los símbolos <>, y las etiquetas de cierre (que indican el final de un determinado contenido) están precedidas por una barra </>.


El Modelo Basico

<!DOCTYPE html> Esta etiqueta declara que tipo de documento vamos a crear, siempre sera la primera etiqueta de nuestro documento y siempre ira en mayusculas la parte del DOCTYPE.
<html> Esta etiqueta declara que a partir de aqui todo lo que escribamos sera HTML.
<head> Esta etiqueta crea una zona en nuestro documento llamada cabecera.
<title> Ejemplo 2 </title> Esta etiqueta se usa para poner un nombre o un titulo a nuestra web.
</head> Esta es la etiqueta de cierre del HEAD o encabezado.
<body> Esta etiqueta abre otra parte de nuestra web llamada body o cuerpo.
</body> Esta etiqueta cierra la etiqueta body.
</html> Esta etiqueta cierra la etiqueta HTML anteriormente abierta.

Divs

<div></div>
Los divs o contenedores son bloques que se suelen usar para distribuir mejor los datos:
Cosas Importantes:
-Comienzan en una línea nueva.
-Ocupan todo el ancho disponible, aunque con css se puede modificar.
Pueden contener otros divs o spans.

Span

<span></span>
 Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.
 Cosas Importantes:
 -Comienzan en la misma línea.
 -Sólo ocupan el espacio necesario.
 -Sólo pueden contener otros spans.


Etiquetas con atributos

Los hiperenlaces o anclas:

Las solemos usar para crear vinculos a sitios.
Formula simple:
<a></a>Sigue el procedimiento habitual.
Ahora vamos a añadirle atributos:
<a name="anclaje">lugar</a> Con el atributo name se referencia esta ancla a otra.
<a href="anclaje">aquí</a> Con el atributo href se enlaza, en el caso de que le demos click a este hipervinculo en el navegador nos llevara a la ancla que tenga como atributo name="anclaje".

El uso de ID:
<etiqueta id="etiqueta"> Contenido </etiqueta> En una etiqueta cualquiera se pone el atributo id para darle un nombre,
<a href="#etiqueta"> Nombre ancla</a> Para luego desde la ancla hacer un enlace entonces cuando le demos a click en el navegador
nos redigira a la etiqueta que tenga ese id.

Otras cosas que se pueden hacer:
Descargar archivos
<a href="url del archivo a descargar" download="Nombre del archivo"> Descargar Archivo</a>

Redireccionar a paginas web, a otras paginas html etc.
<a href="url de la web o pagina html"> Nombre de la pagina</a>
